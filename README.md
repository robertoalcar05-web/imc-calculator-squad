# IMC Calculator

Clone este projeto no GitHub

Depois do fork, clone o repositório em sua máquina usando o:

```sh
git clone ...
```

Com o repositório clonado, instale as dependências:

```sh
npm install
```

Com as dependências instaladas, execute o projeto:

```sh
npm run dev
```

## Exercício

Altere as funções `calculate` e `categorize` no arquivo `main.ts` para que exiba o cálculo do IMC, bem como a categorização

* **Exercício adicional**:

- O que acontece se eu não tiver definido os dois campos ou um deles?
